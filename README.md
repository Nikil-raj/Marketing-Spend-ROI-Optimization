# 📈 Marketing Spend & ROI Optimization Dashboard

## 📌 Project Overview

Marketing teams invest significant budgets across multiple advertising campaigns but often struggle to determine which campaigns deliver the highest return on investment.

This project analyzes real-world marketing campaign data to evaluate campaign performance, optimize advertising spend, and identify high-performing customer segments using SQL, Python, and Power BI.

---

# 🎯 Business Problem

A marketing team wants to answer the following questions:

- Which campaigns generate the highest ROI?
- Which audience segments convert the best?
- Which marketing channels deserve additional investment?
- Where is advertising budget being wasted?
- How can campaign performance be improved?

---

# 🎯 Objectives

- Analyze campaign performance
- Measure advertising effectiveness
- Calculate marketing KPIs
- Identify profitable customer segments
- Build an executive dashboard
- Recommend budget optimization strategies
